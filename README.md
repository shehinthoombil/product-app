# Product App

## Features
✔ Product list page  
✔ Search by title  
✔ Sort by price  
✔ Product details page  
✔ Loading state  
✔ 404 page  
✔ Responsive layout  

### Tech Used
- React + Vite
- Axios
- React Router DOM
- Tailwind CSS

## How to Run
1. git clone 
2. cd product-app
3. npm install
4. npm run dev
